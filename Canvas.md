🎨 Interactive Drawing Canvas
A lightweight, browser-based drawing application built with pure HTML5 and JavaScript. No libraries, no frameworks — just open the file and start drawing.

📋 Description
Interactive Drawing Canvas is a simple yet fully functional drawing tool that runs directly in the browser. It allows users to draw freehand, add shapes, pick colors, adjust brush sizes, and clear the canvas — all without any installation or setup.

✨ Features

✏️ Freehand Drawing — Draw freely on the canvas with a smooth pencil tool
▭ Rectangle Tool — Click and drag to draw rectangles with a semi-transparent fill
○ Circle / Ellipse Tool — Click and drag to draw circles and ellipses
Line Tool — Draw straight lines at any angle
🎨 Color Picker — Choose any color for your brush or shapes
📏 Brush Size Slider — Adjust stroke thickness from 1px to 40px
🗑️ Clear Button — Wipe the entire canvas clean with one click
📱 Touch Support — Works on mobile and tablet devices

Here are all the tools and languages used in this project:

Languages
1. HTML5

Used to structure the webpage
The <canvas> tag is an HTML5 element
The toolbar buttons, color picker, and slider are all HTML elements

2. CSS3

Used to style the toolbar, buttons, and layout
Makes the page look clean and centered
Handles button hover effects and active states

3. JavaScript (Vanilla)

The main language doing all the work
Handles all drawing logic, mouse tracking, shape rendering, and button actions
No external JavaScript library used — pure plain JavaScript only


Browser API
Canvas API

Built into every modern browser
ctx.stroke() — draws lines
ctx.fillRect() — fills shapes
ctx.clearRect() — clears the canvas
ctx.getImageData() — saves a snapshot of the canvas
ctx.putImageData() — restores the snapshot (used for shape preview)
